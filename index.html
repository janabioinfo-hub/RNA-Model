RNA-seq XGBoost Classification

Machine learning pipeline for endometrial tissue phase prediction from RNA-seq data

Launch Colab Notebook GitHub Repository

Features

  • Upload and analyze new test RNA-seq data via drag-and-drop in Colab.
  • Results saved to Google Drive under unique job folders for reproducibility.
  • PCA plots and predictions downloadable for each analysis job.
  • Cloud-friendly setup, all code versioned on GitHub.

How to Use

  1. Open the Colab notebook (see link above).
  2. Run all cells, and drag & drop a CSV file when prompted.
  3. After analysis, download output files using the provided links.
  4. Review your job's results in your Drive under /MyDrive/RNA_Seq_Jobs.
